# MCP Server for Agent-Browser

Model Context Protocol server providing complete browser automation capabilities with token-efficient @ref system.

## Installation

```bash
npm install mcp-server-agent-browser
```

## Quick Start

```bash
mcp-server-agent-browser
```

## Features

- 40+ tools covering all agent-browser functionality
- Token-efficient @ref system
- Complete Playwright API coverage
- Session management
- State persistence
- Network interception
- Video recording
- And more...
